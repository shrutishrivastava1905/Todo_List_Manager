# 📋 To-Do List Manager (C# Console App)

A simple and beginner-friendly **To-Do List application** built using C#. This project allows users to manage daily tasks directly from the console with persistent file storage.

---

## 🚀 Features

* ✅ Add new tasks
* 📄 View all tasks
* ❌ Delete tasks
* 💾 Automatic save to file
* 🔄 Load tasks on startup

---

## 🛠️ Technologies Used

* C#
* .NET Console Application
* File Handling (System.IO)

---

## ▶️ How to Run

### 🔹 Using Visual Studio

1. Open project in Visual Studio
2. Run the project (F5 or ▶ button)

### 🔹 Using .NET CLI

```bash
dotnet new console -n TodoApp
cd TodoApp
dotnet run
```

---

## 💡 How It Works

1. On startup, the app loads tasks from `tasks.txt`
2. Displays a menu to the user
3. User can add, view, or delete tasks
4. On exit, tasks are saved automatically
